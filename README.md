Připravil jsi Vagrant prostředí s Debianem 12.



Pomocí provisioning skriptu jsi automaticky nainstaloval a nastavil Zabbix 7.0 LTS – včetně Apache, PHP, MariaDB, Zabbix serveru a agenta2.



Upravil jsi konfiguraci databáze a soubor zabbix.conf.php, aby se webové rozhraní spustilo bez ručního průvodce a bylo podepsané tvým názvem serveru.



Ověřil jsi běh služeb přes logy, příkazy ps, htop, btop a systemctl status.



Nakonec jsi pořídil snímky obrazovky běžících procesů a webového rozhraní a uložil je do složky ./Images v repozitáři.
